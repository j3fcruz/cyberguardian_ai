# 🧠 Overview

**CyberGuardian AI** is an intelligent, offline cybersecurity assistant designed to function as a **real-time security analyst on your system**.

It continuously observes system behavior and transforms raw technical data into actionable intelligence using an embedded AI reasoning engine.

It monitors:

* Running processes
* Network connections
* Open ports
* File integrity
* System anomalies

Then translates findings into **clear, human-readable security insights**.

No cloud. No telemetry. No external dependencies.

# ❓ Frequently Asked Questions

### Does CyberGuardian AI require internet?

No. It is fully offline and operates locally.

---

### Is my system data uploaded anywhere?

No. All analysis happens on-device.

---

### Does it replace antivirus software?

It complements security tools by providing behavioral and AI-driven analysis, not signature-based antivirus scanning.

---

### Can beginners use it?

Yes. It translates technical threats into plain language explanations.

---

### Does it slow down the system?

It is optimized for lightweight background monitoring with configurable intensity levels.

# 🔧 Troubleshooting

## High CPU Usage

Reduce monitoring frequency in settings.

---

## No Threats Detected

This means either:

* System is clean, or
* Monitoring thresholds are too strict

---

## Network Data Not Showing

Check:

* Firewall permissions
* Admin privileges
* Network adapter access
